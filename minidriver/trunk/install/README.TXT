
Esteid Smart Card Minidriver source

Written for MS v.6.02 specification 
( http://www.microsoft.com/whdc/device/input/smartcard/sc-minidriver.mspx )

External dependencies:
EsteID c++ CardLib
( https://esteidutil.svn.sourceforge.net/svnroot/esteidutil/tags/cardlib ) 
cardmod.h,
Microsoft Windows 2008 DDK

Directory contents:
-
\-esteidcm:    Minidriver code, Visual Studio 2008 Project files
\-cardlib:     c++ lib for Esteid card
\-testac:      command line testing utility, VS2008 Project
\-pkicommon:   utility wrapper classes used by testac
\esteidcm.inf: MS setupAPI install script for the minidriver DLL
\cardmod.h
